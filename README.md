# Práctica 2 – Aprendizaje No Supervisado

## Descripción del proyecto

Este proyecto corresponde a la **Práctica 2 de Aprendizaje Automático**, cuyo objetivo es la aplicación y comparación de distintas **técnicas de aprendizaje no supervisado**, centradas principalmente en **clustering** y **reducción de dimensionalidad**.

A lo largo del notebook se realiza un flujo completo de análisis de datos, desde la carga y exploración inicial hasta la obtención de conclusiones finales, evaluando el comportamiento de distintos algoritmos de agrupamiento sobre un conjunto de datos real.

El trabajo se ha desarrollado en formato **Jupyter Notebook**, facilitando tanto la experimentación como la visualización de resultados.

## Estructura del notebook

El notebook se organiza en las siguientes secciones:

### 1. Introducción
Breve explicación del objetivo de la práctica y de las técnicas que se van a aplicar.

### 2. Carga y exploración de datos
- Lectura del conjunto de datos
- Análisis exploratorio inicial (EDA)
- Identificación de patrones y características relevantes

### 3. Escalado y reducción de dimensionalidad
- Aplicación de distintos escaladores
- Uso de PCA (Principal Component Analysis) para reducir la dimensionalidad
- Visualización de los datos en espacios reducidos

### 4. Técnicas de clustering

Implementación y análisis de distintos algoritmos de agrupamiento:

#### K-Means
- Elección del número óptimo de clusters
- Evaluación de resultados
- Visualización de agrupaciones

#### Clustering jerárquico
- Construcción de dendrogramas
- Comparación entre distintos criterios de enlace

#### DBSCAN
- Ajuste de parámetros
- Identificación de ruido y clusters densos
- Ventajas e inconvenientes frente a otros métodos

### 5. Análisis final y conclusiones
- Comparación de algoritmos
- Discusión de resultados obtenidos
- Conclusiones finales sobre el comportamiento de cada técnica
